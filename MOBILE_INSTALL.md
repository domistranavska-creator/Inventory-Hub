# Mobile Install Guide

Send colleagues the app link and this short message:

## Quick message to send

Open the app link in your mobile browser.

On Android:
- tap the browser menu
- choose `Add to Home screen` or `Install app`
- confirm `Install`

On iPhone:
- open the link in Safari
- tap the `Share` button
- choose `Add to Home Screen`
- tap `Add`

After that, the app will appear on the home screen like a normal app.

## Best way to share it

The easiest setup is:

1. Upload the project to GitHub.
2. Turn on GitHub Pages for the repository.
3. Send colleagues the GitHub Pages link.
4. They install it from the browser to the home screen.

## Recommended text to send colleagues

Open this link on your phone and install it to the home screen.

Android: Menu -> Add to Home screen / Install app
iPhone: Share -> Add to Home Screen

Then it will work like a normal app icon on your phone.
